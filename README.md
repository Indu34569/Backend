# My Backend Project

This is a backend project made using Node.js and Express.
This is beginner-friendly and easy to understand. 

## Features
- Basic API
- CRUD operations

## How to run

Install dependencies:
npm install

Run project:
node index.js

## API Routes
- GET /users
- POST /users
- PUT /users/:id
- DELETE /users/:id
